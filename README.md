# Super Block Jump: Game Boy Advance Edition
An official Game Boy Advance Port of Super Block Jump made by the same creator of the scratch game.

# Controls:
	Gameplay:
		A - Jump
		B - Fire
		Start - Pause
		Select - reverse gravity when accessible
	Pause:
		Start/A: Unpause
		Select/B: Restart level
	Menu:
		Start: begin game
		Start: Show controls
	Titlescreen:
		Start: Begin
